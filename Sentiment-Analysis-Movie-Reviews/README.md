# Sentiment Analysis on Movie Reviews

## Project Overview
This project performs sentiment analysis on movie reviews using Natural Language Processing (NLP) techniques to classify reviews as positive or negative.

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn

## Dataset
IMDb movie reviews dataset from NLTK.

## Methodology
- Text preprocessing (tokenization, stopword removal)
- TF-IDF vectorization
- Model training using Naive Bayes
- Performance evaluation

## Results
- Achieved approximately 80% accuracy on test data
